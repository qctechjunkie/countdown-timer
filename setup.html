Configurable Countdown Timer

This tool has been created for conference sessions, creating an easy to read display of remaining-time for speakers. Timer will display full screen.

You may configure the timer to your liking with the below form. Once configure click "Start Timer"

Timer Options

Countdown start time in minutes and seconds
Updated timer every X seconds
Change to warning at X minutes and seconds left
Updated warning timer every X seconds
Change to alert color at X minutes and seconds left
Updated alert timer every X seconds

Appearance Options

Normal timer colors
Warning colors
Alert colors

Note: Timer can be restarted by simply refreshing the page.
